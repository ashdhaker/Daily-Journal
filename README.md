# Daily-Journal
A Node js Based Application

----> to run the app
first clone this repo
then open in your local system
open terminal 
run command --> "node" 

###### Provide users with a convenient and intuitive platform for daily journaling and reflection

## Features

- Add/make as many blogs you want and it contain heading and
paragraph and show on timeline
- Technologies used - HTML, CSS, JavaScript, Bootstrap, Node.js ,
Express.js , ejs .

### Technologies Used
-  HTML
-  CSS
-  JavaScript
-  Bootstrap
-  Node.js 
-  Express.js
